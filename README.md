# Job Market & Salary Analysis Dashboard

## Overview
This project analyzes job market trends and salary insights using real-world job posting data.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib / Plotly
- Power BI

## Key Steps
- Data Cleaning & Preprocessing
- Feature Engineering (salary ranges, experience levels)
- Exploratory Data Analysis (EDA)
- Exporting cleaned dataset for Power BI dashboard

## Dashboard Insights
- Top in-demand job roles
- Salary distribution across locations
- Most required skills
- Experience vs salary trends

## Output Files
- cleaned_jobs_data.csv (Power BI ready)
- Power BI interactive dashboard (.pbix)
